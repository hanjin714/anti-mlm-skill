# 安装说明

## 环境要求

- Python 3.8+
- pip 包管理器

## 基础安装

```bash
# 克隆仓库
git clone https://github.com/hanjin714/anti-mlm-skill.git
cd anti-mlm-skill

# 安装基础依赖
pip install beautifulsoup4 requests
```

## 可选依赖（增强功能）

```bash
# OCR图片识别（推荐安装）
pip install paddleocr paddlepaddle

# 或使用EasyOCR
pip install easyocr

# 或使用Tesseract（需另外安装tesseract命令）
pip install pytesseract pillow
brew install tesseract  # macOS
```

## 验证安装

```bash
python main.py "测试文本：某大师带你日入过万"
```

输出类似以下内容表示安装成功：
```
🔍 反割韭菜分析报告
📊 风险评分: XX/100
⚠️  风险等级: 高风险
```

## OpenClaw集成

1. 打开OpenClaw工具
2. 导入 `skill.json` 文件
3. 或直接复制 `skill_prompt.py` 中的 `ANALYSIS_PROMPT_TEMPLATE` 作为系统提示词

## 依赖说明

| 依赖 | 用途 | 是否必须 |
|------|------|----------|
| beautifulsoup4 | 网页解析 | 基础功能 |
| requests | HTTP请求 | 基础功能 |
| paddleocr | 图片文字识别 | 可选 |
| easyocr | 图片文字识别 | 可选 |
| pytesseract | 图片文字识别 | 可选 |

## 故障排除

**问题**: `ModuleNotFoundError: No module named 'bs4'`
**解决**: `pip install beautifulsoup4`

**问题**: OCR功能不可用
**解决**: 安装任一OCR引擎，推荐 `pip install paddleocr paddlepaddle`

**问题**: 中文显示乱码
**解决**: 确保终端编码设置为UTF-8
