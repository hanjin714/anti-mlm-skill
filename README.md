帮助用户识别割韭菜陷阱，**主动推荐优质免费学习资源**，真正实现"帮用户避坑+省钱"。

---

## OpenClaw安装

### 方式一：GitHub直接安装

在OpenClaw中执行：
```
/skill install https://github.com/hanjin714/anti-mlm-skill
```

或使用简化命令：
```
/skill install anti-mlm-skill
```

### 方式二：手动导入

1. 下载本仓库的 `skill.json` 文件
2. 在OpenClaw中导入：设置 → 技能管理 → 导入技能 → 选择 `skill.json`

### 方式三：复制Prompt

1. 复制本仓库的 `skill_prompt.py` 中的 `ANALYSIS_PROMPT_TEMPLATE`
2. 在OpenClaw中创建自定义技能，粘贴作为系统提示词

## Python快速使用

```python
from skill_prompt import RiskScorer, PriceAnalyzer, FreeResourceRecommender

# 风险评分
scorer = RiskScorer()
result = scorer.score_text("课程文案...")

# 价格分析
analyzer = PriceAnalyzer()
price_result = analyzer.analyze_price(4999, "编程开发", "beginner")

# 免费资源推荐
recommender = FreeResourceRecommender()
free_result = recommender.recommend("我想学Python")
```

## 分析流程

1. **接收输入** → 文本/图片/链接
2. **风险分析** → 关键词检测、计算评分
3. **价格分析** → 市场价对比
4. **推荐免费资源** → 匹配免费课程
5. **询问用户情况** → 了解收入、目的
6. **综合评估** → 个性化建议

## 红旗信号

出现以下任一特征，**立即远离**：

- 收取入门费/加盟费
- 承诺躺赚、日入过万
