# 🔍 反割韭菜Skill

> 帮你识别课程骗局，主动推荐免费学习资源，省钱又避坑。

---

## ⭐ 特性

- 🤖 **AI智能分析** - 12大场景 + 关键词检测 + 红旗识别
- 💰 **价格匹配** - 结合你的收入，判断课程值不值
- 📚 **免费推荐** - 自动推荐B站/慕课网/官方学院免费课程
- 🛡️ **弱势保护** - 对低收入/急切用户更加谨慎

---

## 📦 安装

### OpenClaw 一键安装

```
/skill install https://github.com/hanjin714/anti-mlm-skill
```

### 手动导入

1. 下载本仓库的 `skill.json`
2. OpenClaw → 设置 → 技能管理 → 导入技能 → 选择文件

### 命令行使用

```bash
git clone https://github.com/hanjin714/anti-mlm-skill.git
cd anti-mlm-skill
pip install beautifulsoup4 requests
python main.py text "某大师带你轻松月入十万"
```

## 🚀 使用

### 发送内容即可分析

支持 **文本**、**图片**、**链接** 三种输入：

```
发送：课程宣传文案/海报截图/课程链接
AI自动分析并返回报告
```

### 分析维度

| 维度 | 说明 |
|------|------|
| 风险评分 | 0-100分，越高越危险 |
| 价格分析 | 检测到的价格 vs 市场合理价 |
| 消费匹配 | 结合你的收入，判断是否承受得起 |
| 免费推荐 | 同类免费课程推荐 |

### 风险等级

| 等级 | 分数 | 建议 |
|------|------|------|
| 🔴 极高风险 | 75-100 | 立即远离 |
| 🟠 高风险 | 50-74 | 高度警惕 |
| 🟡 中等风险 | 25-49 | 谨慎考虑 |
| 🟢 较低风险 | 0-24 | 理性对待 |
