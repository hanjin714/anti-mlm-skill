# 使用说明

## 快速开始

### 方式一：命令行分析

```bash
# 分析文本
python main.py text "好消息！某导师带你轻松月入十万，限时特价只要999！"

# 分析URL
python main.py url "https://example.com/course"

# 分析图片
python main.py image "./poster.jpg"
```

### 方式二：Python代码调用

```python
from skill_prompt import RiskScorer, PriceAnalyzer, FreeResourceRecommender

# 1. 风险评分
scorer = RiskScorer()
result = scorer.score_text("课程宣传文案...")
print(f"风险评分: {result['score']}/100")
print(f"风险等级: {result['level']}")

# 2. 价格分析
analyzer = PriceAnalyzer()
price_result = analyzer.analyze_price(4999, "编程开发", "beginner")
print(f"价格评估: {price_result['assessment']}")

# 3. 免费资源推荐
recommender = FreeResourceRecommender()
free_result = recommender.recommend("我想学Python")
if free_result['recommended']:
    for course in free_result['courses']:
        print(f"推荐: {course['name']} - {course['platform']}")

# 4. 用户消费匹配
afford = analyzer.match_user_affordability(4999, monthly_income=4500)
print(f"承受能力: {afford['match']}")
print(f"建议: {afford['advice']}")
```

### 方式三：OpenClaw多模态分析

1. 导入 `skill.json` 到OpenClaw
2. 发送课程宣传内容（文本/图片/链接）
3. AI自动分析并返回报告

---

## 输入类型

### 文本输入

直接发送课程宣传文案：

```
好消息！某大师亲授，普通人也能轻松月入十万，
限时特价只要999，名额有限，错过等一年！
```

### 图片输入

发送课程海报/截图，AI会自动：
1. OCR识别图片文字
2. 提取价格、联系方式等信息
3. 进行风险分析

### 链接输入

发送课程页面链接，AI会自动：
1. 访问页面获取内容
2. 提取课程信息
3. 核查公司/讲师背景

## 分析维度

| 维度 | 说明 |
|------|------|
| 场景识别 | 识别12大割韭菜场景 |
| 风险评分 | 0-100分评分 |
| 价格分析 | 市场价对比、溢价检测 |
| 消费匹配 | 结合用户收入给出建议 |
| 免费推荐 | 自动推荐官方免费资源 |
| 红旗检测 | 识别拉人头等违法特征 |

---

## 风险等级

| 等级 | 分数 | 建议 |
|------|------|------|
| 极高风险 | 75-100 | 立即远离 |
| 高风险 | 50-74 | 高度警惕 |
| 中等风险 | 25-49 | 谨慎考虑 |
| 较低风险 | 0-24 | 理性对待 |

## 主动询问（可选）

如果想获得更精准的分析，可以主动提供：

1. **您的月收入？** → 用于消费能力匹配
2. **您为什么想学这个？** → 了解学习目的
3. **您之前了解过这个领域吗？** → 评估经验水平

---

## 注意事项

1. 分析结果仅供参考，不能替代专业判断
2. 对于疑似传销的内容，请立即远离并报警
3. 免费资源推荐优先选择官方平台
4. 保持理性，不要被夸张宣传冲昏头脑
